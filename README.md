# Contact Form with Validation

This is a simple and responsive contact form built using HTML, CSS, and JavaScript. It checks if all fields are filled before submitting. The form uses [Web3Forms](https://web3forms.com) to send data without needing a backend.

- HTML form with input fields for name, email, number, and message
- JavaScript validation for empty fields
- Error messages shown below each field
- Success message shown when form is valid
- External CSS file used for styling
- Sends data using Web3Forms API
